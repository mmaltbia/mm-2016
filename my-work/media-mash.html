Hi-Res Photo Search Engine

Designers and developers know the great need we have for high quality license free images to use in our projects.

The Challenge

To develop a lightweight, high resolution, license free photo search experience.

The Solution

I developed a layout that is inuitive and user friendly. Check it out here.

The Result

The site is mobile responsive and shows an array of photos utilizing the Flickr API.

The project also features a way to add photos to an API that I created for use in another project.
